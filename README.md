# 🎓 Campus Lost & Found Portal

## 📌 Overview
The Campus Lost & Found Portal is a Java-based desktop application designed to help students and staff report, search, and claim lost or found items within a campus.

This system replaces manual methods with a simple digital solution, improving efficiency and reducing confusion.

---

## 🚀 Features

- 🔐 User Authentication (Sign In / Sign Up)
- 📦 Report Lost Items
- 📦 Report Found Items
- 🔍 Search Items by keyword
- ✅ Mark Items as Claimed
- 👤 Profile Management (Change Password, Logout)
- 💾 Data Persistence using File Handling
- 🎨 Interactive GUI with basic animations

---

## 🛠️ Technology Stack

- **Language:** Core Java  
- **GUI:** Java Swing & AWT  
- **Concepts Used:** OOP, Event Handling, Collections  
- **Storage:** File Handling (users.txt, items.txt)  
- **Future Scope:** MySQL (via JDBC)

---

## ⚙️ How It Works

1. User registers or logs into the system  
2. User can report lost or found items with details  
3. Items are stored and displayed in a list  
4. Users can search items easily  
5. Claimed items can be removed from the list  
6. Profile allows password change and logout
